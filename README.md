# BridgeCircle

BridgeCircle is an early concept for a verified alumni and community network focused on referrals, mentorship, recruiting, and local connection.

## Repository Contents

- `alumni-network-investor-mvp-pitch.html`: a single-file investor concept and MVP pitch deck
- `project-summary.md`: product thesis, MVP scope, market framing, and growth direction
