Side by side

ThemeBest when…Watch out
1. LinkedInLowest cognitive friction. Users instantly read it as "professional connections." Closest analog to your product thesis.Risk of looking like a LinkedIn-clone. Undersells the warm-network differentiation.
2. SlackDistinctive corporate. Aubergine is rare; reads premium without being flashy."Workplace tool" energy — may feel transactional for an alumni community.
3. NotionYou want UI to disappear. Calm, opinion-free, lets data lead.No emotional hook. Could feel like an internal admin tool.
4. Apple / iOSYou want premium minimalist. Big radii + iOS blue = "polished consumer product."Generic-feels-everywhere risk. The blue is in a thousand other apps.
5. GitHubYou want "well-engineered" energy. Green CTA is unusual and reads action-oriented.Tonally dev-ish. May confuse non-technical board members ("why does this look like a code site?").
6. StripeYou want investors / board to read "ambitious tech startup." Indigo + gradient = premium fintech.Wrong register for a warm-network thesis. Feels transactional.
7. Modern MonochromeMaximum confidence, zero brand commitment. Lets your features speak.Cold. Feels "AI startup," not "alumni community."
8. AirbnbYou want the warmest, most community-feeling option here. Coral + soft shadows = welcoming.Coral may read consumer-leisure rather than professional. "Are we booking a stay?"
9. DiscordYou're betting on a younger / casual demographic. Blurple is friendly + social.Polarizing for a board demo. Reads gaming/teen if not balanced.
10. SubstackYou want the most distinctive option here. Serif + orange = editorial, memorable.Serif body text reads slower. Polarizing for younger alumni who associate serifs with "old."

My ranking for an alumni-network for a school: Apple (#4) > LinkedIn (#1) > Notion (#3) > Airbnb (#8). Apple wins for "premium but unspecific" (your users won't say "this looks Apple," they'll say "this looks polished"). LinkedIn second for direct-thesis fit. Skip Stripe / GitHub / Substack / Modern Mono unless you have a strong contrarian reason.

Inspiration sources

  • Each theme's exact hex codes are pulled from the brand's actual website / public design system. No invented palettes.
  • tweakcn.com — visual shadcn theme editor; gallery of community-built themes. Best place to compare presets and export :root blocks directly.
  • ui.shadcn.com/themes — shadcn's first-party theme picker (10 base palettes, light/dark variants).
  • awesome-shadcn-themes on GitHub — curated list of community theme presets.
  • For institutional alumni products specifically, popular references: Common App's redesign, Handshake (alumni networking platform), and the Yale / Princeton alumni portal redesigns from 2022-2024 — all lean clean monochrome + accent blue, which is what informed picks #1, #4, #7.
  • Brand sources verified: linkedin.com, slack.com, notion.so, apple.com, github.com (Primer Design System docs), stripe.com, linear.app, vercel.com, airbnb.com (DLS public docs), discord.com, substack.com.
