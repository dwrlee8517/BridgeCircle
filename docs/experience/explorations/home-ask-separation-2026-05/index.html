Home direction

  • Compact greeting, not a search hero.
  • Dark priority panel creates a distinct memory from Ask.
  • Rows and queues replace repeated helper cards.
  • Motion represents live network pulse, not matching workflow.

Ask direction

  • Owns the large prompt and animated command edge.
  • Search is the main object, not one module among many.
  • Result cards are allowed because comparison is the workflow.
  • Sidebar stays utilitarian and specific to matching.
