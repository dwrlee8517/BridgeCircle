Font exploration · May 24 · Claude

Type that sounds like BridgeCircle.

The current display face (Fraunces) leans playful and retro — its WONK axis adds curl that can read as "fashion magazine" rather than "trusted alumni network." This document compares the current pairing with four free alternatives, rendered on the actual UI moments (headline, AskBar, person card, status pills, pull-quote) so the difference is visible in context, not in isolation.

What the brand should sound like in type

  • Considered, not trendy. Choose a face that will still feel right in 5 years.
  • Institutional warmth. Closer to a school's alumni magazine than to a chat app. Readers expect care.
  • Readable across generations. The pilot includes Class of '05 (and earlier) members — high x-height + restrained contrast helps.
  • Civic, durable. Avoid faces with strong stylistic signature (excessive curl, quirk axes, display-only quirks).
  • One serif, one sans. Mono stays as JetBrains Mono — it's already restrained.

Adobe's Source family was designed for long-form reading; Serif 4 has a wide spacing and a slightly higher x-height than Newsreader, which makes UI labels and dense rows easier to scan. The pair is naturally tuned. Replacing Inter with Source Sans changes every label in the app. Source Sans is a touch warmer and more humanist than Inter — pleasant, but it shifts the voice slightly toward "civic government" (FCC, HHS use it). If that's the read you want, this is the strongest pick.

The most "prep-school" of the options — Caslon is the canonical English serif, used by The New Yorker variants and many institutional brands. Reads as durable, almost historical, in a way the others do not. Only two weights (400, 700) and no optical-size axis. Tight x-height makes small UI labels feel a bit thin. Lovely on the wordmark and headlines; can fight you on captions.

Recommendation: Option A — Newsreader + Inter

It moves the brand off the playful WONK of Fraunces without losing the editorial feel. Optical-size axis means one family covers wordmark through caption. Inter stays, so every existing label and button keeps its rhythm. It's the smallest change that solves the stated problem (too curly) while leaving the door open to a more institutional shift (Option B) if the Chadwick brand wants a more civic voice.

Sidebar: if you don't want to switch fonts at all

The cheapest move is to tame Fraunces itself. The curl comes mostly from WONK=1 (its quirk axis). Forcing WONK=0 and SOFT=0 at all sizes gives you a much more sober Fraunces. The wordmark below uses 'WONK' 0, 'SOFT' 0, 'opsz' 30 — Fraunces, but grown up:

BridgeCircle

This won't fully solve "the italic pull-quotes feel calligraphic" — Fraunces italic stays warm-quirky regardless of axes. But for the wordmark and h1 headlines, locking the WONK axis to 0 is a one-line change to app_claude/src/components/.../bc-fraunces CSS class.
