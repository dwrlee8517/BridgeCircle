# Explorations

This is the workspace for design experiments, alternative directions, and future visual or UX trials.

Explorations are intentionally **not canonical**. Do not use files in this folder for current design or build iteration unless the user explicitly names an exploration or asks to promote it.

Promotion path:

1. Start experiments in this folder.
2. Decide what should become active guidance.
3. Move the accepted decision into [`../ui/design-system/`](../ui/design-system/), [`../ux/`](../ux/), or [`../screens/`](../screens/).
4. Update app implementation only after the active guidance is clear.

Current subfolders:

- `civic/` - Civic design experiments not yet promoted into the active design system.
- `home-ask-separation-2026-05/`, `home-radar-feature-directions-2026-05/`, `home-visual-directions-2026-05/`, `typography-2026-05/` - active exploration workspaces from May 2026.
