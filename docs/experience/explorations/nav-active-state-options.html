Active Nav State Mockups

Five quick treatments for the signed-in BridgeCircle navigation. Help is selected in each row so the difference is easy to compare against the live page.

Recommended Option B: active icon tile plus a restrained underline.

A. Soft active pill

Clear and friendly. Good if you want the selected item to read like a mode.

Quiet blue-tint background around the active label.

B. Icon tile + label

Compact, modern, and still editorial. This is the best fit for the current header.

Only the active icon gets a surface treatment; label stays clean.

C. Editorial dot rail

Closest to the current underline, with a more intentional civic publication feel.

Small leading dot plus shortened gradient underline.

D. Page-aware accent

Help uses Emerald because it is the offer/help action. More expressive, slightly riskier.

Active state can shift by page purpose without changing the IA.

E. Raised active surface

Most app-like. Strongest selected state, but heavier than BridgeCircle probably needs.

Subtle bordered chip with shadow; useful if the header needs more weight.
