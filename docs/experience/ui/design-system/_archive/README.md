# Archived Civic Design System refs

These HTML files were the v1 reference for the Civic Editorial system before
the Warm adoption (2026-05-25). They are preserved for visual history but **do
not reflect production tokens**. The current tokens are documented in
[`../tokens.md`](../tokens.md), implemented in `app/src/app/globals.css`, and
demoed in the archived experiment at
[`../../../explorations/_archive/warm-2026-05-adopted/warm-design-system.html`](../../../explorations/_archive/warm-2026-05-adopted/warm-design-system.html).

Specifically these files predate:

- `--radius` bump from 6px to 10px
- Body type bump from 13px to 16px
- Modular 1.25 type scale
- Amber `--cta` color and the `cta` button variant
- `.density-cozy` and `.density-pro` mode classes
- Removal of `mono-xs` (9px) and `accent-plum` from active palette

Do not copy values from these files into production code.
