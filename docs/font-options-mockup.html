Side by side

FontBest when…Watch out
1. InterYou want zero risk. Reads correctly to anyone, scales to any size."Everywhere font" — same as 70% of SaaS apps. Your product won't stand out by typography.
2. ManropeYou want Inter's reliability + a bit more warmth. Sweet spot for community products.Very subtle differentiation; some readers can't tell it apart from Inter.
3. Plus Jakarta SansYou want a "modern startup" cue without going experimental. Strong default for landing pages.Slightly heavier feel — may dominate cards/labels at small sizes.
4. DM SansYou want recognizable character. The lowercase 'a' and numerals have personality.Polarizing. Some find DM Sans's quirks distracting in long body text.
5. OutfitCommunity / social product. Friendliest of the geometric sans options. Pairs very well with Discord theme.Reads slightly informal — may undermine "professional alumni network" framing in board demo.
6. SoraYou want geometric / "tech-forward" look. Numbers stand out — great for analytics dashboards.Cool register; less warm than the network's emotional thesis.
7. SatoshiYou want "looks expensive." Highest perceived premium of any free font here.More setup (next/font/local + .woff2 download). Not on Google Fonts.
8. LatoYou want familiar warmth. Lato has a reputation for "approachable" — the Slack-of-fonts vibe.Looks dated to designers. The 2010s SaaS default.
9. IBM Plex SansYou want institutional-but-not-corporate. Strong fit for school / academic context.Strong personality — every character is identifiable as Plex. Not for everyone.
10. OnestYou want Inter-but-fresher. The "current 2024 hotness" pick.Newer; less battle-tested in long-running products.

My ranked picks for BridgeCircle: Manrope (#2) > Plus Jakarta Sans (#3) > Outfit (#5) > Inter (#1). Manrope wins because it's "Inter but warmer" — keeps professionalism, adds the human-network feel. Outfit is the most Discord-aligned (rounder, friendlier) but maybe too informal for the board demo. Skip Lato unless you specifically want Slack's vibe; skip Sora if numbers/density isn't a priority.

How to apply your pick

Three lines of code in app/src/app/layout.tsx + a CSS variable rename. Reversible in one commit.

// 1. Swap the font import in app/src/app/layout.tsx
import { Manrope } from 'next/font/google'   // or whichever you picked

const sans = Manrope({ subsets: ['latin'], variable: '--font-sans' })

// 2. Apply on <html>
<html className={`${sans.variable} font-sans`}>

// 3. Confirm globals.css points to the right variable (already does):
@theme inline {
  --font-sans: var(--font-sans);
  --font-heading: var(--font-sans);
}

If you also want a serif heading font (e.g. Spectral or Playfair for editorial moments), wire a second import to --font-heading and use the font-heading Tailwind utility on <h1> / <h2>.
